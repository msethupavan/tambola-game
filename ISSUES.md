# 🐛 Issues & Bug Reports

## Before Reporting an Issue

- [ ] Check the [existing issues](https://github.com/msethupavan/tambola-game/issues) to avoid duplicates
- [ ] Check the [troubleshooting guide](README.md#-troubleshooting) in README
- [ ] Update Node.js, npm, and all dependencies to latest versions
- [ ] Clear browser cache and try again
- [ ] Try on a different browser or device

## Reporting a Bug

When reporting a bug, please provide:

### 1. **Environment Details**
- Operating System (Windows, macOS, Linux)
- Browser and version
- Node.js version (`node --version`)
- npm version (`npm --version`)
- Network setup (WiFi/LAN network name/type)

### 2. **Steps to Reproduce**
```
1. [First step]
2. [Second step]
3. [Expected behavior]
4. [Actual behavior]
```

### 3. **Screenshots or Videos**
Include visual evidence of the issue if possible.

### 4. **Error Messages**
Copy complete error messages from:
- Browser console (F12 → Console tab)
- Terminal/npm output

### 5. **Additional Context**
- How many players are connected?
- What room code format were you using?
- Have you modified any code?
- Does the issue occur with the build or dev server?

## Feature Requests

Feature requests are welcome! Please provide:

- **Clear title and description**
- **Use case**: Why do you need this feature?
- **Examples**: How would this work in practice?
- **Related issues**: Link to any related issues
